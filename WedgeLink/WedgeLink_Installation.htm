Welcome to WedgeLink

Thank you for choosing WedgeLink.  If you have purchased a WedgeLink Registration ID, follow the procedures in WedgeLink Purchase Installation.  If you will be using WedgeLink in the Preview/Evaluation Mode, follow the procedures in WedgeLink Preview & 3-Day Evaluation Installation.

To avoid possible installation issues, be sure you are logged in with Administrative Rights prior to installing WedgeLink.  Once WedgeLink is installed, you can be logged in with Administrative or User Rights.

WedgeLink Preview & 3-Day Evaluation Installation

Follow these procedures to install WedgeLink in the Preview & 3-day Evaluation Mode.

  • Click Install WedgeLink.  If you get some security dialogs, click Run on these dialogs.

  • You should initially run WedgeLink in the Preview Mode.  Your Evaluation period does not start when you are running in the Preview Mode, not when you install WedgeLink.

  • Your 3-day Evaluation period will start when click the Start Evaluation button (and enter the phrase START EVALUATION).

WedgeLink Purchase Installation

Follow these procedures to install and register WedgeLink if you have purchased a Registration ID for this computer.  You must have a unique Registration ID for each computer that WedgeLink is installed on.

  • Click Install WedgeLink.  If you get some security dialogs, click Run on these dialogs.

  • After installation, run WedgeLink and enter your Registration ID.

  • Be sure to retain this Registration ID in case you have to reinstall WedgeLink on your computer.  If the Registration ID is lost, there will be a charge to replace the Registration ID.

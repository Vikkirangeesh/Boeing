WedgeLink Software Wedges - Version 3

WedgeLink can accept input data from any RS232 serial device, and send that data to any Windows application in such a manner that the application thinks the data is actually coming from the keyboard.  WedgeLink is often used with applications such as Excel, etc.

ComTestSerial

ComTestSerial is a free serial communications test program from MicroRidge Systems.  ComTestSerial will allow you to easily determine what is being sent from a serial device.  You will find ComTestSerial is much easier to use than Microsoft HyperTerminal.

A copy of ComTestSerial will be installed with WedgeLink.  You can also install this test program on other computers to assist you with testing serial communications.

  • Install ComTestSerial on your PC

  • Run ComTestSerial from CD
